# Bluetooth Receiver for Music Assistant

Turn your Home Assistant host into a Bluetooth speaker.

Audio path:

Phone / Laptop
-> Bluetooth A2DP Sink
-> Home Assistant Add-on
-> Music Assistant

## Features

- Bluetooth pairing mode
- Auto reconnect trusted devices
- HTTP stream output
- Direct ingest URL support
- ARM / x86 support

## Install

Add this repo to Home Assistant Add-on Store.
Install **Bluetooth Receiver for Music Assistant**.
